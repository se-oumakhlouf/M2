    </br>

    4. $\exists aAmi . (Campus \sqcap \exists attrape.\{Rougeole\}) \sqsubseteq Confiné$
        - Le concept $Campus$ désigne l'ensemble des personnes dans le campus
        - Le rôle $aAmi$, $aAmi(x, y)$ désigne le fait que $x$ est ami avec $y$
        - Le singleton $Rougeole$ désigne l'ensemble des singletons
        - Le concept $Confiné$ désigne l'ensemble des personnes confiné
        - Le rôle $attrape$, $attrape(x, y)$ désigne le fait que $x$ attrape $y$

    </br>

    5. $Personne \sqcap \neg \{Mary\} \sqsubseteq \exists aime.\{Mary\}$

- ## Exercice 3 :

    1.  - $\Delta^I = \{a^I, b^I\}$
            - $a^I = a$
            - $b^I = b$
            - $A^I = \{a^I,b^I\}$
            - $B^I = \{b^I\}$
            - $C^I = \{b^I\}$
            - $R^I = \{(a^I,b^I),(b^I,b^I)\}$

        </br>

        - $(\exists R.C)^I = \{a^I , b^I\}$
        - $(\forall R.A)^I = \{a^I, b^I\}$
        - $(\exists R.C \sqcap \forall R.A)^I = \{a^I , b^I\}$

        </br>

        - I modèle de KB1 ? 
            - $A^I \subseteq (\exists R.C \sqcap \forall R.A)^I $
            - $\Leftrightarrow \{a^I,b^I\} \subseteq \{a^I , b^I\} \implies I \models \exists R.C \sqcap \forall R.A$
            - $A^I = \{a^I,b^I\} \implies A(a) \implies I \models A(a) $

        - Comme $I$ satisfait tous les axiomes et les assertions alors $I$ est un modele de la $KB1$

    </br>

    2.  - $\Delta^I = \{a^I, b^I\}$
            - $a^I = a$
            - $b^I = b$
            - $A^I = \{a^I ,b^I\}$
            - $B^I = \{b^I\}$
            - $C^I = \{b^I\}$
            - $R^I = \{(a^I, b^I), (b^I, b^I)\}$

        </br>

        - Axiome $1$ : $A \sqsubseteq \exists R.C \sqcap \forall R.A$
            - $(\exists R.C)^I = \{a^I , b^I\}$
            - $(\forall R.A)^I = \{a^I, b^I\}$
            - $(\exists R.C \sqcap \forall R.A)^I = \{a^I , b^I\}$
            - $\implies$ l'axiome $1$ est satisfait

        </br>

        - Axiome $2$ : $A \sqsubseteq \neg C$
            - $(\neg C)^I = \Delta^I \setminus C^I = \{a^I, b^I\} \setminus \{b^I\} = \{a^I\}$
            - $A^I \subseteq (\neg C)^I \Leftrightarrow \{a^I, b^I\} \subseteq \{a^I\}$
            - Ceci est FAUX (car $b^I \in A^I$ mais $b^I \notin \{a^I\}$)

        </br>

        - Comme $I$ ne satisfait pas l'axiome $A \sqsubseteq \neg C$, alors $I$ n'est pas un modèle de $KB_2$

    </br>

    3.  - $\Delta^I = \{a^I, b^I\}$
            - $a^I = a$$b^I = b$
            - $A^I = \{a^I, b^I\}$
            - $C^I = \{a^I, b^I\}$
            - $R^I = \{(a^I, b^I), (b^I, b^I)\}$
        
        </br>
        
        - Axiome $1$ : $A \sqsubseteq \exists R.A$
            - $(\exists R.A)^I = \{a^I, b^I\}$
            - $A^I \subseteq (\exists R.A)^I \Leftrightarrow \{a^I, b^I\} \subseteq \{a^I, b^I\}$
            - $\implies$ L'axiome est satisfait.
            
        </br>
        
        - Axiome $2$ : $\exists R.C \sqcap \forall R.A \sqsubseteq \exists R.A$
            - $(\exists R.C)^I = \{a^I, b^I\}$ (car $a \to b \in C$ et $b \to b \in C$)
            - $(\forall R.A)^I = \{a^I, b^I\}$ (car toutes les flèches mènent vers des $A$)
            - Partie Gauche $(\exists R.C \sqcap \forall R.A)^I = \{a^I, b^I\} \cap \{a^I, b^I\} = \{a^I, b^I\}$
            - Partie Droite $(\exists R.A)^I = \{a^I, b^I\}$$\{a^I, b^I\} \subseteq \{a^I, b^I\}$
            - $\implies$ L'axiome est satisfait.
            
        </br>
        
        - Axiome $3$ : $A \sqsubseteq C$
            - $A^I = \{a^I, b^I\}$
            - $C^I = \{a^I, b^I\}$
            - $\{a^I, b^I\} \subseteq \{a^I, b^I\}$
            - $\implies$ L'axiome est satisfait.
            
        </br>
        
        - $I$ satisfait tous les axiomes et les assertions, donc $I$ est un modèle de $KB_3$.
